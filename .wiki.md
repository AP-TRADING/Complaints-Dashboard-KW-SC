# Project Summary
The KWSC Complaint Dashboard is a web application designed for real-time monitoring and analytics of complaints related to water and sewerage services. Its objective is to enhance operational efficiency and customer satisfaction by providing stakeholders with visual insights into critical metrics, trends, and alerts, thereby facilitating informed decision-making.

# Project Module Description
The project consists of several functional modules:
- **Dashboard**: Optimized for large screens, showcasing key metrics and visualizations.
- **TVMetricCard**: Displays key metrics with trend indicators and color-coded alerts, now bifurcated into "Water" and "Sewerage".
- **Key Insights**: Highlights critical metrics and insights requiring immediate action, with improved font visibility.
- **Critical Table**: Lists towns with critical complaint statuses, now including water and sewerage breakdowns, using conditional formatting for urgency.
- **Monthly Trend Chart**: Visualizes monthly complaint trends, now distinguishing between water and sewerage complaints with distinct colors.
